## Description
This Blender addon streamlines working with collections, objects, and export settings. It automatically distributes objects into Low/High poly collections, assigns proper names, and provides export configuration tools.

## Features
- Automatic distribution of objects into collections (Low/High poly)  
- Assigns proper names to objects  
- Configures FBX export with Blender's standard settings preserved  
- Additional tools for efficient scene management  

## Installation
1. Download the repository or `.zip` file.  
2. In Blender, go to `Edit > Preferences > Add-ons > Install...`  
3. Select the `.zip` file and enable the addon.

## Usage
- Select objects and use the addon panel to distribute them into collections.  
- Export settings can be adjusted in the addon panel within the Scene properties.
